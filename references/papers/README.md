# Publications scientifiques de référence

> Coller ici (ou linker) les papers où Avicenna est auteur, co-auteur, ou
> qui évaluent un produit Avicenna.
>
> Format recommandé :
> - PDF du paper : `<auteur>_<année>_<short-title>.pdf`
> - Fichier metadata associé : `<auteur>_<année>_<short-title>.md` avec :
>   - DOI
>   - Citation complète
>   - Résumé en 5 lignes
>   - Claims citables (et leur formulation autorisée par Reg Affairs)
>   - Limites mentionnées par les auteurs
>
> Ces fichiers sont consultés par les skills blog-article et regulatory-comm
> dès qu'un claim de performance est en jeu.
