# Posts LinkedIn de référence

> Coller ici les 5-15 posts LinkedIn d'Avicenna qui ont le mieux marché
> (engagement, qualité de leads, retours qualitatifs). Un fichier `.md` par
> post avec le contenu + métadonnées.
>
> Format de nom : `YYYY-MM-DD_<account>_<topic-court>.md`
>
> Ex : `2025-11-12_corporate_clearance-cina-lvo.md`
>
> Métadonnées à inclure en header de chaque fichier :
> - compte (corporate / Cyril / etc.)
> - audience visée
> - performance (impressions, engagement rate, comments quality)
> - pourquoi ça a marché (1-2 lignes d'analyse)
>
> Ces fichiers sont chargés par le skill linkedin-post comme références
> de style.
