# Prompt — Annonce d'une recrue stratégique

## Quand l'utiliser

Pour annoncer publiquement l'arrivée d'un profil senior (C-level, VP, Chief
Medical Officer, etc.). Pour les arrivées « régulières », un simple post
équipe suffit (utiliser directement le skill linkedin-post).

## Prompt

```
On annonce l'arrivée de [Nom Prénom] en tant que [Titre]. Voici son profil :

- Parcours : [bullet points]
- Pourquoi on est ravis : [3 raisons]
- Ce qu'iel va piloter chez Avicenna : [scope]
- Citation Cyril préparée : [oui / à drafter]
- Citation de la recrue : [oui / à demander]
- Photo dispo : [oui / non]
- LinkedIn de la personne : [URL]

Charge brand pack et produis :

1. Post LinkedIn corporate (voix Avicenna)
2. Post LinkedIn CEO Cyril (voix personnelle, pourquoi je l'ai recrutée)
3. Section newsletter du mois (« nouveau visage chez Avicenna »)
4. Slack interne (annonce all-hands chaleureuse)
5. (optionnel) Mini-interview 4 questions pour blog si profil très
   différenciant (ex : Chief AI Officer venant d'un labo prestigieux)

Vérifications spécifiques :
- Pas de chiffre RH confidentiel (salaire, equity)
- Pas de comparaison désobligeante avec d'autres employeurs
- Pas de promesse roadmap implicite (« va lancer X » → seulement si X est
  déjà annoncé publiquement)
- Si la personne vient d'un concurrent direct, traiter avec soin (pas de
  triomphalisme, focus sur ce qu'elle apporte, pas sur ce qu'elle quitte)
```
