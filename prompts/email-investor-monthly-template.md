# Template — Investor monthly update (ultra-court)

## Quand l'utiliser

Pour le mensuel investisseurs / board / advisors. Format VC-friendly : 5 minutes à lire, pas plus.

## Prompt à coller

```
Charge brand/proof-points.md (chiffres internes du mois).

Génère un investor update mensuel pour [mois] [année].

Format strict : un seul email court, structure ci-dessous.

=== EMAIL ===

Subject : Avicenna — [Mois Année] update

Hi all,

[1-2 phrases tone-setter — un fait marquant du mois, une perspective, ou
une thèse, pas un "ça avance bien"]

📈 KPIs du mois
- [KPI 1] : [valeur] ([variation vs mois précédent])
- [KPI 2] : ...
- [KPI 3] : ...
(Max 5 KPIs. Si pas de variation pertinente, skip le KPI ce mois.)

✅ Wins
- ...
- ...
(Max 3, faits, sans hype)

🚧 Challenges / Risks
- ...
- ...
(Max 2-3. Honnête. Les investisseurs préfèrent un risque nommé qu'un risque
caché.)

🎯 Next 30 days focus
- ...
- ...
(Max 3 priorités)

🙏 Asks
- [Intro à X souhaitée]
- [Conseil sur Y bienvenu]
- [Aide sur recrutement Z]
(Si rien, écrire "Rien ce mois-ci" plutôt que d'inventer.)

Best,
Cyril

=== FIN EMAIL ===

Voici les inputs du mois (à transformer en items courts) :

[Coller les bullets bruts du mois]
```

## Anti-patterns à éviter

- Pavé de texte sans structure
- Hype ("on est sur une trajectoire incroyable")
- Pas de chiffres → pas crédible
- Pas de "asks" → message implicite "tout va bien, pas besoin de vous"
- Confondre l'update avec un pitch deck (c'est un signal fréquence/transparence, pas une vente)
